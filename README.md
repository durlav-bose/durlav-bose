## Render Engine
Ultra-realistic product mockup generation platform capable of **50K+ renders per day**.

• Photoshop-level canvas distortion (warp / puppet warp)  
• AI-powered prompt based background & model generation  
• Reduced rendering time **40s → <10s**  
• Built with **AWS Lambda, Puppeteer, Redis**

A **Shopify-like multi-tenant eCommerce platform** built for Bangladeshi merchants.

• Subscription SaaS model

---
